This project represents a solution for the Ex2 assignment. 
It implements a spreadsheet application capable of saving strings, numbers, and formulas in cells.
The application can perform simple arithmetic operations (+, -, /, *) on the formulas provided.
Features:
-Store strings, numbers, and formulas in cells.
-Evaluate arithmetic expressions.
-Retrieve the value of any cell.
-Showes error whenever an invalide formula is enterd.
